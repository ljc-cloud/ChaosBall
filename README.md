# ChaosBall
**基于原神小游戏雀球开发的小游戏**

### 1.游戏场景
![alt text](img/game.png)

### 2.游戏玩法
1. 每人有四次投球，投进圈中得分（红-4,黄-2,蓝-1）
2. 超时会放弃本次投球，并切换投球权
3. 可以将其他人的球撞出得分区
